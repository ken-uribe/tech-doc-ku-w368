Technical Ghost-Hunting Guide

Introduction

This documentation provides a complete reference for ghost-hunting operations in Luigi's Haunted Mansion. From capturing Boos to defeating portrait-class bosses, everything is technically broken down for field agents and developers.

Equipment Overview

The following tools are standard issue for ghost-hunting ops:

  • Poltergust G-00: Multifunctional vacuum for ghost capture.
  • Dark-Light Device: Reveals hidden objects and ghosts.
  • Gooigi: Slime-based Luigi clone for inaccessible areas.
  • Virtual Boo: Communication and system logging device.
  • Strobulb: High-intensity flashbulb for stunning enemies.
Poltergust G-00 API

The Poltergust G-00 is programmable. Below is an example call to activate suction mode:


Poltergust.activateMode("suction");
Poltergust.setIntensity(0.85); // Suction power from 0.0 to 1.0
Poltergust.target("ghostType", "Greenie");
Poltergust.execute();

Ghost Capture Mechanics

Ghost capture consists of three phases:

  • Stun: Using the Strobulb to immobilize.
  • Suction: Engage Poltergust and begin resistance meter balancing.
  • Slam: Build energy to slam ghosts for damage amplification.
Environmental Interactions

Environmental tools that assist in ghost-hunting:

  • Ceiling fans: Reveal hidden compartments.
  • Curtains: Often hide Boos or keys.
  • Toilets: Surprisingly, frequent ghost spawn point.
  • Mirrors: Used for quick teleportation.
  • Vents: Gooigi access points.
Boss Battle Framework

Each boss follows a typical three-stage lifecycle:

  1. Pattern Recognition: Observe attack patterns.
  2. Exploit Trigger: Use object/environment to disable shield or armor.
  3. Ghost Phase: Boss becomes vulnerable for capture or damage.
Slime Luigi Deployment (Gooigi)

Gooigi can be summoned when normal traversal fails. Be aware of water hazards.


Gooigi.deploy();
Gooigi.navigateTo("ventilationShaft");
Gooigi.recall();
Advanced Capture Combos

Combining stun and slam allows for crowd control:


const map = MansionMap.load("F3_EastWing");
map.rooms.forEach(room => {
  if (room.containsGhosts) {
    GhostRadar.mark(room);
  }
});
Common Ghost Classifications

Ghosts are categorized by behavior and strength:

  • Greenies: Basic melee attackers.
  • Hiders: Hide in furniture, appear when scanned.
  • Slammers: High HP, slow movement.
  • Slinkers: Interrupt player controls.
  • Boos: Named ghosts with unique traits.
Boss Weakness Exploits

Each boss has a coded vulnerability logic:


function bossIsVulnerable(boss) {
  return boss.isStunned() && !boss.hasShield();
}
if (bossIsVulnerable(currentBoss)) {
  Poltergust.capture(currentBoss);
}
Mansion Map Parsing

Mansion blueprints are loaded dynamically. To parse map segments:


const map = MansionMap.load("F3_EastWing");
map.rooms.forEach(room => {
  if (room.containsGhosts) {
    GhostRadar.mark(room);
  }
});
Multiplayer Ghost Syncing

When playing in co-op mode, follow sync protocol:

  • Host player initiates sync session.
  • Secondary players validate their Gooigi link.
  • All players calibrate Strobulb brightness.
  • Enemy health pools are shared.
  • Slam combo bonuses stack per player.
Troubleshooting & Known Bugs (List Section)

Known issues and fixes:

  • Gooigi freezes near water: Expected behavior. Avoid wet areas.
  • Boss phase loops infinitely: Reset phase counter via debug menu.
  • Camera jitter in elevator scenes: Patch pending.
  • Ghost clipping through floor: Toggle collision layer.
  • Suction not registering: Reboot Poltergust API.
